# repos

Add this repository in the POM.xml
```xml
<repositories>
  <repository>
    <id>my-github</id>
    <url>https://maven.pkg.github.com/uia4j/repos</url>
    <snapshots>
      <enabled>true</enabled>
    </snapshots>
    <releases>
      <enabled>true</enabled>
    </releases>  
  </repository>
</repositories>
```
